

# AUREA - Handmade Jewelry E-Commerce Website 💎

AUREA is a modern and elegant front-end e-commerce website designed for a handmade jewelry brand. The project provides a complete shopping experience where users can explore jewelry products, view product details, add items to their cart, and manage their shopping cart.

The website focuses on a clean luxury design, smooth user interaction, and a simple shopping flow.

---

## ✨ Features

- Responsive jewelry store design
- Modern product showcase
- Product listing page
- Product detail page
- Related products section
- Add to cart functionality
- Cart management system
- Increase/decrease product quantity
- Remove products from cart
- Dynamic cart item counter
- Add to cart toast notifications
- User-friendly navigation
- Elegant jewelry-inspired UI

---

## 🛒 Shopping Cart System

The cart system is created using JavaScript and LocalStorage.

Users can:

- Add products to cart
- View selected products
- Change product quantity
- Remove products
- See updated totals
- Keep cart data saved after refreshing the page
